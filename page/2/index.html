    3.跨域

    jsonP跨域：通过src进行跨域
    function aa(data){console.log(data)}

    或jquery中直接dataType:”jsonp”就可以跨域

    $.ajax({
    url:请求的地址,
    method:”get”,
    async:true,
    dataType:”jsonp”
    success:function(value){},
    error:function(value){}
    })
    jsonp请求原理：
    因为用ajax跨域请求时，会有问题，但是在src这个地址中不存在跨域，所以把要请求的地址写到src中，然后在地址后面用&callback=aa，
    表示使用自定义的函数aa去包裹请求到的数据，然后在aa这个函数中用形参去接收返回来的数据
    上述原理在jQuery封装的ajax中，直接使用dataType:”jsonp”表示
    跨域请求数据的方式：
    ①代理：使用服务器代理
    ②XHR2：在ie10以下不兼容，设置请求头
    ③jsonp：通过src没有跨域问题

    6.字符串

    ①被引号包裹的内容就是字符串
    ②使用new Sring 创建字符串（对象）
    字符访问：
    var str = “nihao”;
    str.charAt(下标);
    str.charCodeAt(下标); //102 字符编码
    字符串比较：比较编码值,从第一位字符依次比较,如果第一位比出结果后，后面就不用比了
    位置方法：字符串的下标：str.indexOf(“要查找的字符”，要查找的起始下标);
    查找的字符不存在 返回值为-1
    str.lastIndexOf();从字符串的后面向前查找，第一次查找到的字符下标，下标还是从第一位开始
    截取方法：str.slice(起始下标,终点下标); 起始下标会被截取到，终点下标不会被截取到（slice会将负数和字符串的长度相加）
    str.substring(起始下标，终点下标);起始下标会被截取到，终点下标不会被截取到（会根据两个参数的大小，自动调整起点下标和终点下标，substring负数转化为0）
    切割方法：str.split(切割字符);返回由指定的分割字符 切割目标字符串后得到的字符串构成的数组
    str.split(); 如果不写切割字符则返回由全部字符构成的长度唯一的数组
    str.split(“”); 返回所有单个字符构成的数组
    str.split(“”，3); 限定返回数组的项数（默认全部返回）
    大小写转换：切换大写 str.toUpperCase();
    切换小写 str.toLowerCase();
    str.trim()去掉字符串前面和后面的空格 //ie9+
    替换方法：str.replace(需要被替换的内容，新内容); 只能替换第一个字符
