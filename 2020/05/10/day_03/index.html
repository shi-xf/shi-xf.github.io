1.其他类型转为String

String()
变量.toString() 括号里不用填东西

2.流程语句

顺序控制 从上到下执行
条件语句 if if…else if..else if..else… switch
循环语句 for.. for..in.. while.. do..while
其他 break continue
①if(条件){在条件满足时执行}
可以省略{},只能执行紧挨着最近的一条语句
②switch(t){
case 情况1:语句1;break;
case 情况2:语句2;break;
……
default:语句
}
case为t可能的所有值
break不能省略，如果没有break则从满足条件的语句开始执行后面所有的语句
switch语句使用于变量值为简单的数字或者符号
③for循环语法：
for(声明变量;结束循环条件;改变变量){循环体}

3.其他获取元素的方式

①通过标签名获取元素
document.getElementsByTagName(“标签名”);
获取出来的为数组，具体某个值用下标获取
②通过类名获取元素
document.getElementsByClassName(“类名”);
获取出来的为数组，具体某个值用下标获取
③通过父元素获取子元素
先是把父元素获取到，然后
父元素.getElementsByTagName(“子元素标签名”);
父元素.getElementsByClassName(“子元素类名”);

4.双层for循环

外面的循环循环一次，里面的循环要循环完
可以看成表格，行为外面的循环，列为里面的循环
