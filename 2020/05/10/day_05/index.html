4.使用函数

参数：
实参：实际的参数
形参：形式上的参数
function fun(a,b){} 形参 var a a代表10,b代表20;
fun(10,20);//实参
①当形参个数与实参个数不一样时，多余的实参不参与计算
②当形参个数与实参个数不一样时，多余的行参undefined
③实参可以是任意类型，一般不传null和undefined，没有意义
function fun(a){
a(); 代表执行回调函数
}
fun(function(){console.log(1)}); 回调函数

function fun1(a,b,c,e,f){
arguments; 函数中的内置对象，类似数组，arguments中的参数个数由实参决定，形参可以不用写
}
fun1(1,2,3,4,5,6);

6.函数的作用域

作用域：变量或者函数的使用范围
作用域链：变量或者函数的查找机制，从内向外查找 自己–父–全局
