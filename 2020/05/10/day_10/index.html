1.表单获取元素

先获取到form元素
通过name值来得到表单元素
同一个name获取的是数组类型，需要加下标

2.表单事件

form.onsubmit = function(){}
return true 表单继续提交
return false 阻止表单继续提交
form.onreset = function(){}
confirm() 带选择项的对话框 点击确定：返回true 取消：返回false
form.元素.onfocus = function(){} 表单元素获取焦点
form.元素.onblur = function(){} 表单元素失去焦点

3.表单操作

表单外的元素点击提交表单内容
form.submit(); 触发表单提交事件
form.reset(); 触发表单重置事件
form.元素.focus() 元素获取焦点的方法
form.元素.select() 元素里面的内容被选中的事件

4.BOM

BOM：window就是BOM的核心
①提供了操作浏览器相关的接口（API）
②js的顶层对象 global object，定义的全局变量会自动注册为window的属性，函数变成window的方法

5.BOM的窗口

alert() 警告弹窗
window.confirm() 确认弹窗（确认提示信息）点击确定：返回true 取消：返回false
window.prompt(“提示信息”)带输入框的弹窗（确认提示信息，默认字符串） 点击确定：返回输入文本 取消：返回null

6.open和close

open() 打开页面（url，打开方式（_blank默认,新窗口打开,_self，当前窗口打开），新窗口的一些配置项）
例：open(“网址”,”_self”,”width=300，height=300”); 打开一个新页面,_self代表在本页面打开
行间使用open：不能省掉window
close() 关闭当前窗口

7.location

location:保存用户的地址栏信息，同时也有一定的导航功能
完整的url：location.href
location = “网址” 页面跳转
主机信息：host = hostName+port
location.hostName：主机名（IP地址） location.port：端口号
location.pathName：文件路径
location.protocol：协议
location.hast（哈希值）
location.search（查询内容）

8.navigator

navigator:保存用户代理信息

9.history

history:保存用户历史访问信息，历史窗口访问
history.go(-1) 退后一个页面
history.go(1)前进一个页面

10.client

clientWidth = width+padding 内部容器的宽
clientHight 内部容器的高
clientLeft 左边框的宽度
clientTop 上边框的宽度
document.documentElement.clientWidth 窗口可视区域的宽
document.documentElement.clientHight 窗口可视区域的高

11.offset

offsetWidth width+padding+border 元素的占位宽度
offsetHight 元素的占位高度
offsetLeft 距离定位参考元素的左偏移值
offsetTop 距离定位参考元素的上偏移值

12.scroll

元素内部内容的实际宽高
scrollWidth
scrollHight
元素.onscroll = function(){}
scrollLeft 内容被卷动到左侧的部分宽度值
scrollTop 内容被卷动到上侧的部分高度值
window.onscroll = function(){
console.log(document.documentElement.scrollTop) //网页内容被卷动到上侧的高度
}

14.BOM事件

onload
onscroll
