1.对象的概念

new String(); 字符串对象
①创建对象 ②访问对象的属性方法 ③销毁对象
内置函数 window的方法

2.日期对象

var date = new Date(); 世界时,当前日期对象
date.getFullYear(); 年
date.getMonth(); 月（1月是0）
date.getDate(); 日期
date.getDay(); 星期（星期日是0）
date.getHours(); 小时
date.getMinutes(); 分
date.getSeconds(); 秒
date.getTime(); 时间戳（从世界时 1970/1/1 00:00:00 距离当前的日期对象的毫秒数）

3.获取特殊时间格式

var date = new Date();
日期字符串：date.toDateString(); Tue Apr 23 2019
当地日期字符串：date.toLocaleDateString(); 2019/4/23
时间字符串：date.toTimeString();中国标准时间
当地时间字符串：date.toLocaleTimeString(); 上午11:11:26

4.设置日期对象

①var date = new Date();
date.setFullYear() 设置年份
setMonth setDay setDate setHours
②给new Date传入参数设置日期
var date = new Date(2019,3,24,12,0,0) //number 年/月+1/日 h:m:s（默认0）
var date1 = new Date(“2019,4,24,12:00:00”);//string 年/月/日 h:m:s（默认0）

5.倒计时

两个时间对象相减为两个时间戳的差值 毫秒数

6.字符串

①被引号包裹的内容就是字符串
②使用new Sring 创建字符串（对象）
字符访问：
var str = “nihao”;
str.charAt(下标);
str.charCodeAt(下标); //102 字符编码
字符串比较：比较编码值,从第一位字符依次比较,如果第一位比出结果后，后面就不用比了
位置方法：字符串的下标：str.indexOf(“要查找的字符”，要查找的起始下标);
查找的字符不存在 返回值为-1
str.lastIndexOf();从字符串的后面向前查找，第一次查找到的字符下标，下标还是从第一位开始
截取方法：str.slice(起始下标,终点下标); 起始下标会被截取到，终点下标不会被截取到（slice会将负数和字符串的长度相加）
str.substring(起始下标，终点下标);起始下标会被截取到，终点下标不会被截取到（会根据两个参数的大小，自动调整起点下标和终点下标，substring负数转化为0）
切割方法：str.split(切割字符);返回由指定的分割字符 切割目标字符串后得到的字符串构成的数组
str.split(); 如果不写切割字符则返回由全部字符构成的长度唯一的数组
str.split(“”); 返回所有单个字符构成的数组
str.split(“”，3); 限定返回数组的项数（默认全部返回）
大小写转换：切换大写 str.toUpperCase();
切换小写 str.toLowerCase();
str.trim()去掉字符串前面和后面的空格 //ie9+
替换方法：str.replace(需要被替换的内容，新内容); 只能替换第一个字符
