1.数组

数组：一系列有序元素的集合
创建数组：①字面量表达
②使用函数 var arr = new Array(1,2,3) 只传入一个数字时，代表数组长度
数组可以储存任意类型数据

2.数组的常用方法

栈方法 LIFO（后进先出），只能在栈顶操作
push()：在数组的尾部添加若干项
返回 数组的新长度
pop()：去除数组的最后一项
返回被去除的项
队列方法 FIFO（先进先出）
shift()：去除数组的第一项
返回被去除的项
unshift()：在数组的头部添加若干项
返回数组的新长度

3.数组的增删改

①增
splice(1,0,”jack”)
1代表添加项起始下标
0代表没有删除的项
“jack”代表被添加的项，可添加多项
返回数组的新长度
②删
splice(1,2);
1代表起始下标
2代表需要被删除的项
返回被删除项构成的数组
③改
splice(1,1,”zhian”);

5.数组的排序

①选择排序
②冒泡排序

6.数组的sort方法

sort();令数组排序，默认升序
比较函数
a，b 任意相邻的两项
sort(function(a,b){
return b-a; 降序（正数）
return a-b; 升序（负数）
})
比较字符串数字用函数比较
Date.parse(“2019-4-23”) 把日期字符串转化为时间戳

7.迭代方法 ie9+

value代表数组中的每一项 index代表当前项的下标
every(function(value,index){
return value>0; //判断每一项是否大于0
})
every：返回每一项是否都满足函数内的条件，都满足返回true，任意一项不满足返回false
some：返回是否有任意一项满足函数内的条件，有一个满足就返回true，都不满足返回false
filter：返回满足条件的项构成的新数组
map：返回一个新数组
forEach：没有返回值，相当于for循环

8.其他方法

reverse(); 逆序
concat(); 返回一个多个数组的每一项拼接的新数组
join() 和split相反，通过指定的字符将数组拼接成字符串
indexOf()
lastIndexOf()

9.Math

Math.PI //3.1415926 圆周率
Math.round(1.1) //1 四舍五入
Math.ceil(1.1) //2 向上取整,有小数就进1
Math.floor(1.9) //1 向下取整,只看整数
Math.pow(2,3) //2的3次幂
Math.aqrt(9)//开方
Math.abs(-1) //绝对值
Math.random() //随机数[0,1)
