day_09

发表于 2020-05-10

1.js的构成

ECMAScript：规定js和核心规则
数据类型、流程语句、解析机制
ECMAScript 3.0 -> ECMAScript5
DOM：document object model 文档对象模型
BOM：browser object model 浏览器对象模型

2.子节点

DOM：描绘html文件内部构成的一个模型（一个树状结构，由层次节点构成）
层次由标签 文本 属性 注释……构成
获取子节点
①父元素.children 获取htmlcollection:类数组结构（标准浏览器使用）
获取子元素 ie6-ie8：会获取注释
②childNodes:nodeList 节点列表
会获取到换行，换行是文本节点
ie9-：不会获取到换行

3.每个节点都有一些属性

①父元素.nodeType:节点类型
3 -> text(文本节点) 1 -> 元素节点 2 -> 属性节点 8 -> 注释节点
②nodeName:节点名
例：LI(大写的标签名) #text文本
③nodeValue:节点值
只有文本节点和注释节点才有

4.父节点

子元素.parentNode 获取父节点
每个节点可以有多个子节点，但是只有一个父节点
offsetParent（定位父节点）找有定位的父元素，一层一层找，最后为body

5.兄弟节点

元素.previousSibling 上一个兄弟节点（注意换行算是文本节点）非标准浏览器不会获取到换行
元素.previousElementSibling 上一个兄弟元素 标准浏览器
元素.nextSibling 下一个兄弟节点（换行是节点）非标准浏览器不会获取到换行
元素.nextElementSibling 下一个兄弟元素 标准浏览器

6.首尾节点

第一个子节点和最后一个子节点
父元素.firstElementChild 第一个元素子节点 标准浏览器
父元素.firstChild 第一个子节点（换行是节点） 非标准浏览器不会获取到换行
父元素.lastElementChild 最后一个元素子节点 标准浏览器
父元素.lastChild 最后一个子节点（换行是节点） 非标准浏览器不会获取到换行

7.创建节点

var cLi = document.createElement(“标签名”) 创建标签
父元素.appendChild(cLi); 作为父元素的最后一个元素来添加
cLi.innerHTML = “串串” 添加内容
var text = document.creatTextNode(“串串”); 创建文本节点
cLi.appendChild(text); 追加进入内容

8.插入节点

只能插入在子元素的前面
父元素.insertBefore(cLi,要插入的子元素)
在父元素内部某个元素前面插入一个元素

9.已存在的元素的插入是改变元素的物理位置

10.删除

删除的元素.remove(); 移除节点
父元素.removeChid(子元素); 移除子节点

11.替换

父元素.replaceChild(新的节点，被替换的节点)

12.克隆

var 变量名 = 需要克隆的元素.cloneNode(true);
父元素.appendChild(变量名)
false(默认)：浅克隆（克隆元素）
true(默认)：深克隆（克隆元素和元素的内部结构）

13.新的获取元素的方法 ie8+

通过选择器语法获取对应的元素：document.querySelectorAll(“父 子”)
通过选择器语法获取对应的元素的第一项：document.querySelector(“父 子”);

14.属性操作

getAttribute(属性名) 获取属性（包括自定义属性）
setAttribute(属性名,属性值) 设置属性（attr,value）

15.表格

tab.tHead 获取thead
tab.tBodies[0] 获取tbody (tBodies 复数项)
tab.tFoot 获取tFoot
tab.tBodies[0].rows[0] 获取tr
tab.tBodies[0].rows[0].cells[1] 获取td
